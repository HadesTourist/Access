# Access

In development - 22/02/2021

 Disability Accessability Chrome extension - Uses DOM manipulation to alter websites to suit the needs of people with disabilities
 
 Features to be aimed for:
 
 1. Ability alter websites to high or low contrast for to aid people to read websites
 2. Highlight and read - Reads selected section of website.
 3. Mute - Mutes all elements on website upon loading.
 4. Alter Text Size - Make reading easier
 5. Brightness - Limit brightness of websites
 6. Button size - alters buttons sizes making clicking easier
 7. Addition of Alexia support to allow blind users to command features on off
 8. User account to select specific feature allowing people to tailor sites to their needs.
